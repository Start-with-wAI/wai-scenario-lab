# Scenario Guide Skill

Use this skill when preparing raw questionnaire answers for the Scenario Lab workflow.

Instructions:
1. Use only the selected scenario configuration and the user's current answers.
2. Convert answers into structured facts without adding unstated context.
3. Mark vague, missing, or contradictory details as missing information.
4. Ask for no more than one clarification when required.
5. Strip obvious personal, account, credential, customer, or confidential details.
6. Do not recommend tools, vendors, automations, or business actions.
