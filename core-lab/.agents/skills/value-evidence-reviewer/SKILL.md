# Value and Evidence Reviewer Skill

Use this skill when choosing the Scenario Brief measurement.

Instructions:
1. Select exactly one measurement from the active scenario configuration.
2. Prefer primary measurement only when required evidence is present.
3. Use the configured fallback measurement when answers are vague or incomplete.
4. Express measurements only in non-financial units such as minutes, incidents, ideas, or attempts.
5. Label evidence strength as strong, partial, or insufficient.
6. Never invent savings, ROI, guarantees, annual value, or dollar projections.
